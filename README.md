# Luthar-task-optimizer
A smart task management app built with JavaSript to track productivity and optimize workflow.

## Features
- Add and delete tasks
- Categorize tasks (Work, Study, Personal)
- Mark tasks as complete
- Real-time statistics (Total, Completed, and Completion Rate)

## Concept
This project reflects structured thinking and optimization - turning simple task tracking into measurable performance.
